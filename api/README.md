## Setup

Install the project dependencies.

```
yarn
```

## Usage

**Start server offline**

```
$ serverless offline start
```

**Deploy**

```
$ serverless deploy
```

**Invoke the function locally.**

```
serverless invoke local --function hello
```

**Invoke the function**

```
curl https://xxxxxxxxx.execute-api.us-east-1.amazonaws.com/
```
